## Environment Variables

To run this project, you will need to add the following environment variables to your .env file:

`NEXT_PUBLIC_API_URL`    
`PORT`  
`MONGO_DB_URI`  
`REFRESH_TOKEN_SECRET`    
`ACCESS_TOKEN_SECRET`  
`CLOUDINARY_CLOUD_NAME`  
`CLOUDINARY_API_KEY`  
`CLOUDINARY_API_SECRET`  
`STATUS`    
`GMAIL_SMTP_USER`  
`GMAIL_SMTP_PASS`
